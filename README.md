# Bolt — Transaction List Color Audit

Color audit of the Bolt transaction list page (Real UI 2025 Design System mapping).

Each color rendered on `bolt.playrealbrokerage.com/transactions/all/active` is mapped against the official Real UI 2025 design tokens — flagged as **Match** (exact), **Mapped to closest** (visually close), or **Not in DS** (needs a new token).

Includes 199 audited colors across:
- Page header, breadcrumb, tabs, sub-tabs, sort bar
- Transaction card (default + expanded states)
- All 21 status pill variants (Termination Requested, Commission Document Generated, etc.)
- Property type pills (Residential, Commercial, Condo, Mobile Home)
- All hover popovers: Road To Success, Checklist, Requires Attention, Compliance Verified, Power Audit
- Filters modal, Sort dropdown
- All 12 tab × sub-tab combinations (All / Sale / Lease / Referrals × Active / Closed / Terminated)
